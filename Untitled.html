HTML Vocab

Website

A location connected to the Internet that maintains one or more pages on the World Wide Web.

HTML

Stands for Hypertext Markup Language and is used to create the content of a web page.

HTML Tag

The special set of characters that tells the machine where the start and end of an HTML element is and what type it is.

HTML Element

A piece of a website. Marked by a start tag and sometimes closed with an end tag. Also includes the content of the elemtn as well.

Website Content

The raw text, images, and other elements included in a web page.

Website Structure

The purpose of different pieces of content in a web page, used to help the computer determine how that content should be displayed.

Header

A title or summary for a document or section of a document.

Digital Footprint

the collected information about an individual across multiple websites on the Internet.

Citation

A quotation from or reference to a book, paper, or author, especially in a scholarly work.

Copyright

The exclusive legal right, given to an originator or an assignee to print, publish, preform, film, or record literary, artistic, or musical material, and to authorize others to do the same.
